# **App Name**: Whitespace

## Core Features:

- Secure Authentication: PIN-based authentication for team members, managed through Supabase.
- Reference Library: Reference Library: Save and organize articles/videos with notes and timestamps.
- AI Reference Suggestions: AI-powered suggestions for related articles/videos based on current project and past saved links. LLM uses saved links as a tool to make decisions.
- Smart Titles: AI-powered title autocompletion for saved links.
- API Key Repository: Secure API Key Repository: Store API keys with masking, tags, notes, and expiration dates, protected by double PIN authentication.
- Smart API Guidance: AI-driven identification of API types with integration guide suggestions. LLM uses saved links as a tool to make decisions.
- Project Management: Projects Module: Track project status, type, and relevant links with AI-powered suggestions for tools, case studies, and prompts.

## Style Guidelines:

- Background color: Off-white (#FAF9F6) for a light, neutral backdrop.
- Primary color: Dark gray (#333333) for a strong contrast on the light background.
- Accent color: Light gray (#808080) to provide visual interest and guide user attention.
- Font: 'Space Mono' (monospace) for all text, providing a consistent, coding-oriented feel.
- Use minimalist icons from Lucide for clear and concise visual representation.
- Implement a simple grid layout for module arrangement and easy navigation.
- Employ smooth microinteractions using Framer Motion to enhance the user experience.
